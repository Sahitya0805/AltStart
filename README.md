# AltStart Website 

A 1:1 pixel-perfect clone of [AltStart](https://altstart.in/) built with clean HTML5, CSS3, Vanilla JS, and Python SPA routing server.

## Features
- **Hero & Services**: Light pastel/vivid yellow gradients, exact official photo assets, checkmark service cards, infinite scrolling partner marquees.
- **About Us**: Complete 5-section layout with team photo assets, philosophy cards, story banner, and statistics.
- **Contact Us**: Responsive contact form with details and soft yellow gradient.
- **Interactive Chatbot**: Floating chatbot widget with quick reply options, auto-responses, and dismissable speech bubble.
- **Calendly Modal**: "Let's Talk" scheduling popup with step-by-step meeting booking.
- **SPA Routing**: Client-side routing with `server.py` fallback for clean URLs (`/our-services`, `/about-us`, `/contact-us`).

## How to Run Locally

```bash
python3 server.py
```

Then open `http://localhost:8080` in your browser.
